# PhagesDB - The Actinobacteriophage Database

## Overview

PhagesDB is a **community database for actinobacteriophages** (phages infecting Actinobacteria like Mycobacterium, Streptomyces). Largest collection of phages from a single bacterial phylum.

**Key Statistics:**
- **Phages Catalogued**: 30,000+
- **Sequenced Genomes**: 5,000+
- **Educational Program**: SEA-PHAGES integration
- **Community-Driven**: Contributions from labs worldwide

## Data Quality

### Strengths
- **Community Curation**: Active user community
- **Educational Integration**: High-quality annotations (students + experts)
- **Local BLAST**: Unpublished sequences available
- **Rich Metadata**: Host, location, discoverer, isolation
- **Comparative Tools**: Built-in genome comparison

### Limitations
- **Actinobacteria Only**: Not comprehensive for all phages
- **Bias**: Heavy Mycobacterium focus
- **Annotation Depth**: Still many hypothetical proteins
- **Limited to Cultured**: No metagenomic phages

## Use Cases

### ✅ GOOD FOR:
- Actinobacteriophage research
- Mycobacterium phages
- Educational projects (SEA-PHAGES)
- Comparative actinophage genomics
- Phage diversity in single phylum

### ❌ NOT GOOD FOR:
- Non-Actinobacteria phages
- Metagenomic phages
- Broad viral diversity

## Access

**Web**: https://phagesdb.org
**BLAST**: Local database includes unpublished
**Download**: Genomes, annotations

## Bottom Line

**Best resource for actinobacteriophages**. If you work with Mycobacterium, Streptomyces, or other Actinobacteria phages, this is essential. Community-driven and educational focus.
