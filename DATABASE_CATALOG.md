# Complete Phage Database Catalog

This document lists ALL major databases containing phage data. Use this to track which have been reviewed.

## Primary Source Databases

### Genome Databases
- [x] **BFVD** - AlphaFold viral protein structures (162K phage proteins)
- [ ] **NCBI Viral Genomes** - Official viral genome repository
- [ ] **IMG/VR** - Metagenomic viral sequences (>15M sequences)
- [ ] **GPD (Gut Phage Database)** - 142K gut phage genomes
- [ ] **MGV (Metagenomic Gut Virus)** - 189K gut viral genomes
- [ ] **GOV2.0** - Global Ocean Viromes (bulk downloads)
- [ ] **GVD (Gut Virome Database)** - Gut viral dark matter

### Protein/Annotation Databases
- [x] **UniProt** - Gold standard protein annotations (SwissProt + TrEMBL)
- [x] **VOGDB** - Virus Orthologous Groups (~400K VOGs)
- [ ] **Pfam** - Protein domain families (viral subset)
- [ ] **InterPro** - Integrated protein signatures
- [ ] **eggNOG** - Orthologous groups (includes viruses)
- [ ] **KEGG Viruses** - Metabolic pathways

### Structure Databases
- [x] **BFVD** - AlphaFold viral proteins (351K structures)
- [ ] **PDB** - Experimental protein structures (~20K viral)
- [ ] **AlphaFold DB** - Individual protein predictions
- [ ] **ESMFold** - Alternative structure predictions

### Prophage Databases
- [ ] **Prophage-DB** - 356K prophage sequences
- [ ] **PHASTER** - Prophage finder web service
- [ ] **PhageWeb** - Prophage identification
- [ ] **VirSorter2** - Viral sequence detection (tool + DB)

### Curated/Reference Databases
- [ ] **InPhared** - Complete phage genomes (consistent annotation)
- [ ] **PhagesDB** - Actinobacteriophages (30K+ phages, 8K entered)
- [ ] **MillardLab Phage-Host Database** - Experimental host ranges

### Metagenomic/Environmental
- [ ] **IMG/VR** - Integrated Microbial Genomes/Virus
- [ ] **GPD** - Gut Phage Database
- [ ] **MGV** - Metagenomic Gut Virus
- [ ] **GOV2.0** - Global Ocean Viromes
- [ ] **GVD** - Gut Virome Database
- [ ] **Serratus** - Planetary virome (petabase-scale)

### CRISPR Databases
- [ ] **CRISPRdb** - CRISPR spacers and repeats
- [ ] **CRISPRCasdb** - CRISPR-Cas systems
- [ ] **CRISPRTarget** - Spacer-protospacer matches

### Specialized/Therapeutic
- [ ] **Phage Directory** - Community phage collections
- [ ] **ATCC Bacteriophages** - Culture collection
- [ ] **DSMZ Phage Collection** - German collection
- [ ] **Félix d'Hérelle Center** - Therapeutic phages

## Annotation Tools (with databases)

### Automated Annotation Pipelines
- [ ] **Pharokka** - Phage annotation pipeline (uses VOGDB, Pfam, etc.)
- [ ] **DRAM-v** - Viral annotation (distillation of metabolism)
- [ ] **CheckV** - Viral genome quality/completeness
- [ ] **VIBRANT** - Virus identification and annotation
- [ ] **VirSorter2** - Viral sequence identification

## Aggregators/Meta-Resources

### Multi-Database Portals
- [ ] **PhageScope** - Aggregates 873K sequences from 10+ databases
- [ ] **PhageDive** - Strain-level data from culture collections
- [ ] **ViralZone** - Viral biology knowledge base (ExPASy)

### Specialized Aggregators
- [ ] **Virus-Host DB** - Host range information
- [ ] **MVP (Microbe vs Phage)** - Phage-microbe interactions
- [ ] **VHRdb (Viral Host Range)** - Curated host range data
